# 💸 Dynamic Pricing Feedback Summarizer

## 📌 Project Overview
This project ingests customer feedback (e.g., complaints about surge pricing) and uses LLMs to summarize them into actionable insights. It helps Uber’s operations, pricing, and product teams better understand user sentiment and pain points around dynamic pricing.

---

## 🧠 Key Features

- ✍️ **LLM Summarization**: Converts free-form complaints into daily summary reports.
- 🔍 **Sentiment & Intent Detection**: Tags issues as positive/negative/neutral and categorizes by theme.
- 📈 **Streamlit Dashboard**: Visualize trends, frequent complaints, and summary stats.
- 🧪 **Supports Multiple Formats**: Works with JSON, CSV, or direct text input.

---

## 🛠️ Tech Stack

| Component   | Technology              |
|-------------|--------------------------|
| Backend     | Python + FastAPI         |
| UI          | Streamlit                |
| LLM         | OpenAI GPT via LangChain |
| Storage     | MongoDB / JSON / CSV     |
